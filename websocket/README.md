# Cours Web temps réel 
Ce répo contient l'architecture de "base" pour suivre le cours de web temps réel. Il permet de lancer simultanément un serveur front et back afin de pouvoir utiliser les websockets. 



## Installer et lancer la correction

Installation
```
npm install
```


Lancer le serveur "back"
```
npm run server:back
```

lancer le serveur "front"
```
npm run server:front
```
